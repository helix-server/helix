# Hello World Example

This is a simple "Hello World" example application built using the Helix Server framework. It demonstrates the basic structure and functionality of a Helix Server application.

## Prerequisites

- [Bun](https://bun.sh) installed on your system

## Installation

```bash
bun install
```

## Running the Application

To run the application, use the following command:

```bash
bun run serve
```

The server will start and listen on [http://127.0.0.1:8080](http://127.0.0.1:8080).
